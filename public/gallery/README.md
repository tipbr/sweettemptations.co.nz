# Gallery Images

Place your bakery gallery images in this directory.

## Image Guidelines:
- Recommended format: JPG or WebP for photos
- Recommended size: 1200px width (will be optimized by Next.js)
- Use descriptive filenames (e.g., `wedding-cake-1.jpg`, `cupcakes-assorted.jpg`)
- Add alt text descriptions in the Gallery component for accessibility

## Example Structure:
```
/public/gallery/
├── wedding-cake-1.jpg
├── wedding-cake-2.jpg
├── birthday-cake-1.jpg
├── cupcakes-assorted.jpg
├── cookies-decorated.jpg
└── custom-cake-example.jpg
```

Once images are added, they will automatically appear in the gallery page.
